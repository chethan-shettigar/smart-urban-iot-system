# Smart Urban IoT System

## Overview
Developed an integrated IoT-based smart city solution to automate and optimize urban infrastructure including irrigation, street lighting, traffic management, and waste segregation. The system is designed to improve efficiency, reduce resource wastage, and enable remote monitoring.

## Modules

### 1. Smart Irrigation System
- Automated watering based on soil moisture levels
- Reduces water wastage through sensor-based control
- Supports remote monitoring and control

### 2. Smart Street Lighting
- Automatic ON/OFF based on ambient light conditions
- Energy-efficient operation
- Scalable for smart city deployment

### 3. Traffic Management System
- Intelligent traffic signal control using sensor inputs
- Adaptive timing to reduce congestion
- Improves traffic flow efficiency

### 4. Automatic Garbage Segregation
- Segregates waste (wet/dry) using sensor-based detection
- Enhances waste management efficiency
- Reduces manual intervention

## Technologies Used
- ESP32 / Microcontrollers
- Embedded C
- GPIO, ADC, and Peripheral Interfaces
- LCD Display (16x2)
- Sensors:
  - Soil Moisture Sensor (Irrigation)
  - Water Level Sensor
  - LDR (Light Sensor for Street Lighting)
  - IR Sensors (Traffic / Detection)
  - Metal Detection Sensor (Garbage Segregation)
- Actuators:
  - DC Motors (Garbage Segregation Mechanism)
  - Relays / Switching Circuits (Street Lights)
- IoT Communication (Blynk Platform)

## IoT Integration (Blynk Platform)
- Integrated Blynk IoT platform for real-time monitoring and control
- Displayed live sensor data on mobile dashboard
- Enabled remote control of irrigation and street lighting systems
- Improved system accessibility through cloud connectivity

## Key Features
- Multi-domain system integration (IoT + Embedded Systems)
- Real-world smart city application
- Scalable and modular architecture
- Remote monitoring and control capability
